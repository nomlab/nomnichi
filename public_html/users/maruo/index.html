研究テーマ

テーマ:
Thread ボーダルータによるネットワーク管理性向上に関する検討
概要:
近年，IoT デバイスの普及にともない，エッジネットワークの信頼性・可用性の低下および電力効率・通信範囲の両立の困難さという2つの問題がある．この問題への対応として Thread と呼ばれる低消費電力なメッシュネットワーク技術が策定された．しかし，Thread にはメッシュネットワークを形成するという特徴から，トポロジが動的に変化することでネットワーク管理が困難になるという課題がある．本研究では，Thread ネットワークと外部ネットワークのブリッジであるボーダルータの機能を拡張し，Thread ネットワークの情報を収集できるようにする．また，それらの情報をユーザが直感的に理解できる形式で可視化することで Thread ネットワークの管理性向上を図る．
