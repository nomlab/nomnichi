研究テーマ

テーマ:
ユーザランド I/O の遅延改善に向けた sched_ext によるスケジューラの試作
概要:
Linux において，専用のデバイスドライバを用いず，ユーザランドの機能で I/O を行うことがある．たとえば，Simple Open EtherCAT Master (SOEM) は， Raw socket を用いることで，デバイスとの通信を行う．このような I/O は， デバイスドライバを用いた I/O 方式と比較して，カーネルバージョンや NIC を問わず利用できるという利点がある．一方で，OS スケジューリングの影響による遅延の発生が問題となる． 本稿ではこの問題に対し，ユーザが作成した BPF プログラムをスケジューラとして利用可能にする機能 sched_ext を用いて，ユーザランド I/O の遅延改善に取り組む．
