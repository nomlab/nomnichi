研究テーマ

テーマ:
マイクロサービス実行基盤wasmCloudの評価
概要:
近年，WebAssembly（Wasm）は，移植性やセキュリティ分離性に優れた実行基盤技術として注目されている． サーバサイドやエッジ環境におけるマイクロサービスの実行基盤としての利用が進んでおり，Wasm を用いたマイクロサービス実行基盤の一つに wasmCloud がある． 一方で，従来広く用いられてきたマイクロサービス実行基盤にはコンテナ オーケストレーションプラットフォームである Kubernetes がある． 本研究では，Wasm を用いたマイクロサービス実行基盤である wasmCloud と，従来の代表的な実行基盤である Kubernetes を対象として，アーキテクチャおよび性能特性の比較評価を行った． まず，マイクロサービス実行基盤の役割を整理し，wasmCloud と Kubernetes のマイクロサービス実行環境および通信方式の特徴を比較した． 次に，Request Per Second (RPS) とレイテンシ分布を指標として性能評価を行い，インスタンス数を変化させた場合の性能特性を測定した．
